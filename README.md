# leakyautofill.github.io
A website for testing the autofill functionality of password managers.

## File Structure
```
├── login-self/         // Login forms with [password <input> fields] hidden using various techniques.
├── cvv-self/           // Credit card forms with [credit card number <input> fields] hidden using various techniques.
├── pii-self/           // Personal information forms with [telephone and address <input> fields] hidden using various techniques (without email field).
├── pii_email-self/     // Personal information forms with [telephone and address <input> fields] hidden using various techniques (with email field).
├── login/              // Login forms with [password <input> fields' ancestor <div> element] hidden using various techniques.
├── cvv/                // Credit card forms with [credit card number <input> fields' ancestor <div> element] hidden using various techniques.
├── pii/                // Personal information forms with [telephone and address <input> fields' ancestor <div> element] hidden using various techniques (without email field).
├── pii_email/          // Personal information forms with [telephone and address <input> fields' ancestor <div> element] hidden using various techniques (with email field).
├── index.html          // Home page.
├── README.md           // README
```
